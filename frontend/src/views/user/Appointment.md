# 教练预约模块技术文档

## 1. 功能概述

教练预约模块是私人健身房管理系统的重要组成部分，允许用户预约教练进行一对一训练。该模块实现了以下核心功能：

- 教练选择功能
- 时间选择器
- 预约记录管理系统
- 预约修改与取消功能

## 2. 技术实现

### 2.1 前端技术栈

- Vue.js 2.6
- Element UI
- CSS3 (响应式设计)

### 2.2 核心组件

#### 2.2.1 教练选择组件

- 实现了教练列表展示，包含教练头像、姓名、专长标签和评分
- 支持点击查看教练详情，包括完整介绍和用户评价
- 显示教练可预约日期，已被预约时段置灰显示

#### 2.2.2 时间选择组件

- 日历式日期选择器，默认显示当前日期及未来7天
- 30分钟为单位的时间段选择
- 时长选择（0.5/1/1.5/2小时）

#### 2.2.3 预约记录组件

- 按时间倒序排列的预约记录列表
- 状态标识（待确认/已确认/已取消）
- 支持按状态和日期范围筛选

#### 2.2.4 预约操作组件

- 预约修改功能
- 预约取消功能（带24小时规则）
- 二次确认弹窗

## 5. 界面设计说明

### 5.1 布局结构

- 顶部：页面标题和操作按钮
- 中部：标签页切换（预约教练/预约记录）
- 左侧：教练列表
- 右侧：时间选择和预约详情

### 5.2 响应式设计

- 桌面端（≥1024px）：教练卡片网格布局，多列显示
- 平板端（768px-1023px）：教练卡片双列布局
- 移动端（≤767px）：教练卡片单列布局，时间槽自适应宽度

### 5.3 交互设计

- 教练卡片：悬停效果，点击选择
- 日期选择：点击高亮，显示可预约状态
- 时间选择：点击高亮，已预约时段置灰
- 操作按钮：加载状态，成功/失败反馈
- 弹窗确认：二次确认，操作后果说明

### 5.4 视觉设计

- 主色调：#409EFF（蓝色）
- 辅助色：#67C23A（绿色，已确认）、#E6A23C（黄色，待确认）、#909399（灰色，已取消）
- 字体：系统默认字体
- 间距：统一的20px间距
- 圆角：4px圆角

## 6. 性能优化

- 数据本地缓存：使用localStorage缓存教练列表和预约记录
- 懒加载：教练头像懒加载
- 防抖：时间选择和筛选操作防抖处理
- 虚拟滚动：预约记录列表使用虚拟滚动
