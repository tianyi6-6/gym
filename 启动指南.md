# 私人健身房管理系统 - 启动指南

## 📋 前置准备

### 1. 检查环境
确保已安装以下软件：
- ✅ **JDK 1.8** - 检查命令：`java -version`
- ✅ **MySQL 5.7 或 8.0** - 检查命令：`mysql --version`
- ✅ **Maven 3.8+** - 检查命令：`mvn -version`
- ✅ **Node.js 16+** - 检查命令：`node -v`
- ✅ **npm** - 检查命令：`npm -v`

---

## 🗄️ 第一步：数据库配置

### 1.1 启动MySQL服务
确保MySQL服务正在运行：
- Windows: 在服务管理器中启动MySQL服务
- Linux/Mac: `sudo systemctl start mysql` 或 `brew services start mysql`

### 1.2 创建数据库
打开MySQL客户端（如Navicat、MySQL Workbench或命令行），执行以下操作：

**方式一：使用命令行**
```bash
mysql -u root -p
# 输入密码后执行
source database/gym.sql
```

**方式二：使用图形化工具**
1. 打开Navicat或MySQL Workbench
2. 连接到MySQL服务器
3. 打开 `database/gym.sql` 文件
4. 执行整个SQL脚本

**方式三：手动执行**
```sql
-- 1. 创建数据库
CREATE DATABASE IF NOT EXISTS gym CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 2. 使用数据库
USE gym;

-- 3. 执行 gym.sql 中的所有表创建语句
```

### 1.3 修改数据库配置
编辑 `backend/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/gym?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root        # 修改为你的MySQL用户名
    password: root        # 修改为你的MySQL密码
```

## 🔧 第二步：启动后端服务

### 2.1 进入后端目录
```bash
cd backend
```

### 2.2 安装依赖（首次运行）
```bash
mvn clean install
```

如果下载依赖较慢，可以使用国内镜像，编辑 `~/.m2/settings.xml`：
```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```

### 2.3 启动Spring Boot应用

**方式一：使用Maven命令（推荐）**
```bash
mvn spring-boot:run
```

**方式二：使用IDE**
1. 使用IntelliJ IDEA或Eclipse打开项目
2. 找到 `GymApplication.java` 文件
3. 右键 -> Run 'GymApplication'

**方式三：打包后运行**
```bash
mvn clean package
java -jar target/gym-backend-1.0.0.jar
```

### 2.4 验证后端启动成功
看到以下日志表示启动成功：
```
Started GymApplication in X.XXX seconds
```

后端服务地址：`http://localhost:8080`
API接口地址：`http://localhost:8080/api`

---

## 🎨 第三步：启动前端服务

### 3.1 进入前端目录
打开**新的终端窗口**（保持后端服务运行），执行：
```bash
cd frontend
```

### 3.2 安装依赖（首次运行）
```bash
npm install
```

如果下载较慢，可以使用淘宝镜像：
```bash
npm install --registry=https://registry.npmmirror.com
```

### 3.3 启动开发服务器
```bash
npm run serve
```

### 3.4 验证前端启动成功
看到以下信息表示启动成功：
```
App running at:
- Local:   http://localhost:8081/
- Network: http://192.168.x.x:8081/
```

## ⚠️ 常见问题解决

### 问题1：数据库连接失败
**错误信息**：`Communications link failure` 或 `Access denied`

**解决方案**：
1. 检查MySQL服务是否启动
2. 检查 `application.yml` 中的数据库配置是否正确
3. 检查MySQL用户权限
4. 确认数据库 `gym` 已创建

### 问题2：端口被占用
**错误信息**：`Port 8080 is already in use`

**解决方案**：
- 修改 `application.yml` 中的端口号：
```yaml
server:
  port: 8081  # 改为其他端口
```

### 问题3：前端无法连接后端
**错误信息**：`Network Error` 或 `CORS error`

**解决方案**：
1. 确认后端服务已启动（访问 `http://localhost:8080/api/auth/login` 测试）
2. 检查 `frontend/vue.config.js` 中的代理配置
3. 确认后端CORS配置正确

### 问题4：Maven依赖下载失败
**解决方案**：
1. 使用国内镜像（见第二步）
2. 清理Maven缓存：`mvn clean`
3. 删除 `~/.m2/repository` 后重新下载

### 问题5：npm install 失败
**解决方案**：
1. 删除 `node_modules` 和 `package-lock.json`
2. 使用淘宝镜像：`npm install --registry=https://registry.npmmirror.com`
3. 检查Node.js版本是否为16+

### 问题6：登录时提示密码错误
**解决方案**：
1. 确认使用的是默认账号：admin/123456
2. 如果修改了数据库密码，需要重新生成BCrypt密码
3. 运行 `PasswordGenerator.java` 生成新密码

## 🔄 日常启动流程

以后每次启动项目，只需：

1. **启动MySQL服务**
2. **启动后端**（在 `backend` 目录）：
   ```bash
   mvn spring-boot:run
   ```
3. **启动前端**（在新终端，`frontend` 目录）：
   ```bash
   npm run serve
   ```
4. **访问**：`http://localhost:8081`
